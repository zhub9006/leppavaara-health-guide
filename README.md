# 🏥💪 Leppävaara Health & Fitness Guide

A community resource guide for Leppävaara residents, listing all health-related businesses and fitness facilities in the area. All locations are sourced from **OpenStreetMap (OSM)** data and cross-verified geocoding.

> **Coverage Area:** Leppävaara and surrounding sub-districts (Etelä-Leppävaara, Pohjois-Leppävaara, Vallikallio, Ruusutorppa) in Espoo, Finland — approximately 3 km radius from Leppävaara center (60.220°N, 24.814°E).

**Leppävaaran terveysasema** — Main health center. GP, nurse visits, preventive care, vaccinations, health screenings.

**Leppävaaran neuvola** — Municipal child health clinic. Growth assessments, vaccinations, parenting support.

**Terveystalo Leppävaara hammaslääkäripalvelut** — Dental services, part of the Terveystalo healthcare network.

## 🤝 Contributing

This is a community project! To add or update entries:
1. **Check OSM first** — If a place exists in OpenStreetMap, add it here with the OSM tags.
2. **Verify addresses** — Use Google Maps or the business's website.
3. **Add categories** — Place each entry in the correct section above.
4. **Include contact info** — Phone, website, and opening hours are very helpful.
5. **Open an Issue or Pull Request** — Propose changes via GitHub.
