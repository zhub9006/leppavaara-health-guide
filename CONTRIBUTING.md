# Contributing to Leppävaara Health & Fitness Guide

Thank you for your interest in improving this community resource! This guide is maintained by volunteers and relies on community contributions to stay accurate and useful.

## How to Contribute

### Adding a New Entry

1. **Search for the place on OpenStreetMap** at [openstreetmap.org](https://www.openstreetmap.org).
2. **Find the OSM tags** — health-related places are typically tagged with:
   - `amenity=pharmacy` / `amenity=health_center` / `amenity=clinic` / `amenity=doctors`
   - `shop=optician` / `shop=chemist` / `shop=health_food`
   - `leisure=fitness_centre` / `leisure=sports_centre` / `leisure=swimming_pool`
3. **Verify the address** — confirm the street name, house number, and postal code.
4. **Geocode the location** — note the WGS84 coordinates (latitude, longitude).
5. **Open an Issue** with the new entry details, or submit a Pull Request.

### Suggested Entry Format

When adding a new entry, please include:

| Field | Example |
|-------|---------|
| Name | **Elixia Sello** |
| OSM type | `leisure=fitness_centre` |
| Address | Leppävaarankatu 3-9, 02600 Espoo |
| Coordinates | 60.2175, 24.8155 |
| Phone | +358 10 309 9430 |
| Website | elixia.fi |
| Notes | Open 24/7; cardio/strength, pool, sauna |

### Categories

Entries in the README are organized by these categories:

- 🏥 Healthcare & Clinics (tertiary/primary care, dental, pharmacies)
- 💪 Fitness & Gyms (indoor gyms, studios, climbing, outdoor stations)
- 🏊 Swimming Pools
- 🏟 Sports Centres & Halls
- 🥗 Health Food Stores
- 📢 Health-Adjacent Shops

### Correction Guidelines

- If an entry has outdated information, update it and note the change.
- If a business has closed, remove it from the active list or mark it clearly.
- Always cross-reference with the OSM node ID when possible.

## Data Sources

- **OpenStreetMap (OSM):** Primary source — all entries include their OSM tag type.
- **Finnish health registries:** For verifying healthcare provider names.
- **Community verification:** Residents confirm newly added entries.
