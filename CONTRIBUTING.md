# Contributing to Leppävaara Health & Fitness Guide

Thank you for your interest in improving this community resource! This guide relies on OpenStreetMap (OSM) data, so every addition helps make it more accurate and comprehensive.

## How to Add or Correct an Entry

1. **Search OpenStreetMap** — Visit [openstreetmap.org](https://www.openstreetmap.org) and search for the business or facility you want to add.
2. **Verify the OSM tags** — Check that the facility has the correct tags:
   - Pharmacies: `amenity=pharmacy` or `shop=pharmacy`
   - Clinics/Health centers: `amenity=clinic`, `healthcare=clinic`
   - Gyms/Fitness: `leisure=fitness_centre`, `gym=fitness_centre`
   - Swimming pools: `leisure=sports_centre` + sport tags
   - Health food stores: `shop=health_food`
3. **Geocode the address** — Confirm the address and coordinates match OSM data.
4. **Follow the format** — Use the table format already in the README.
5. **Open a Pull Request** — Submit your changes with a clear description.

## Data Sources

- Primary: **OpenStreetMap** (osm.org) — all entries mapped to OSM features
- Verification: Geocoding via Nominatim with coordinate cross-checking
- Distance estimates: Straight-line from Leppävaara centre (60.220°N, 24.814°E)

Thank you for helping keep our community guide accurate and useful! 🏥💪
