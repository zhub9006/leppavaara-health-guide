# 🤝 Contributing to the Leppävaara Health Guide

Thank you for your interest in improving this community resource guide!

## How to contribute

### Adding a new business
1. Fork the repository
2. Edit `README.md` — add your entry in the appropriate section
3. Include: **name, address, and website/phone** if known
4. Make sure your data is verifiable (OSM tag, official website, or phone book)
5. Open a Pull Request with a clear description

### Reporting outdated information
1. Open a **GitHub Issue** describing what needs updating
2. Include the corrected information and your source if possible

### Data sources we use
- **OpenStreetMap** — the primary data source for all locations
- **Sello shopping centre directory** — [sello.fi](https://www.sello.fi)
- **Espoo city services** — [espoo.fi](https://www.espoo.fi)
- Local verification (walking around, visiting businesses)

### Formatting guidelines
- Keep tables sorted alphabetically by venue name
- Include links to official websites where available
- Add OSM coordinates in the Area Reference table for new locations
- Use the same emoji prefix as existing sections

## Community rules
- All entries must be **legitimate, operating businesses**
- No promotional or self-listed businesses (unless you are the business owner)
- Please be respectful in discussions

Happy mapping! 🗺️
