# 🤝 Contributing to the Leppävaara Health Guide

Thank you for your interest in improving this community resource guide!

## How to contribute

### Adding a new business
1. Fork the repository
2. Edit `README.md` — add your entry in the appropriate section with **name, type, address, and OSM-verified coordinates**
3. Include links to official websites where available
4. Verify your data on [OpenStreetMap](https://www.openstreetmap.org/)
5. Open a Pull Request with a clear description

### Reporting outdated information
1. Open a **GitHub Issue** describing what needs updating
2. Include the corrected information and your source if possible

### Data sources we use
- **OpenStreetMap** — the primary data source for all locations (verified via geocoding & reverse geocoding)
- **Sello shopping centre directory** — [sello.fi](https://www.sello.fi)
- **Espoo city services** — [espoo.fi](https://www.espoo.fi)
- Local verification (visiting businesses)

### Formatting guidelines
- Keep tables sorted alphabetically by venue name within each section
- Include **type**, **address**, and **OSM coordinates** (✅) where available
- Use the same emoji prefix as existing section headers
- Don't duplicate entries — update existing ones instead

### Adding new categories
1. Add a new section in the Table of Contents
2. Add the corresponding header in the README
3. Include a brief description and a table with columns appropriate to the category
4. Note the OSM tag types (e.g., `shop=supermarket`, `leisure=fitness_centre`)

## Community rules
- All entries must be **legitimate, operating businesses** in the Leppävaara area
- No promotional or self-listed businesses (unless you are the business owner)
- Please be respectful in discussions
- Keep the guide useful for **residents**, not businesses

## Getting started
- **Primary Health Center**: Leppävaaran terveysasema (Konstaapelinkatu 2)
- **Health cluster**: Hevosenkenkä 3 / Ruusutorppa (Terveystalo, Mehiläinen, Apteekki Elixir)
- **Fitness**: Elixia (Leppävaarankatu, Sello)
- **Swimming**: Leppävaaran uimahalli (Veräjäpellonkatu 15)
- **Sports hall**: Leppävaaran urheiluhalli (Veräjänpieli)
